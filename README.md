# 🎄 Christmas Face Filter Mini-Game

## 🌟 프로젝트 소개
본 프로젝트는 Python과 MediaPipe를 활용하여 사용자의 얼굴 움직임(입 벌림, 머리 움직임)을 인식하여 플레이하는 실시간 크리스마스 테마 미니 게임입니다. 단순한 필터를 넘어, 상호작용 가능한 엔터테인먼트 경험을 제공하여 텀 프로젝트의 창의성과 완성도를 높였습니다.

## 🛠️ 기술 스택
* **언어:** Python
* **핵심 라이브러리:** MediaPipe, OpenCV
* **추가:** NumPy, Pygame (렌더링 용)
